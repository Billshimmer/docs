## 分支管理

> git的分支功能是核心功能，大型项目多人协同开发的场景下，就需要依赖分支管理来实现并行开发和功能管理。

## 命名规范

> 在git开发中的命名主要针对commit、branch, 下面是一些比较常见的例子:

>
```
git add . && git commit -m '添加文案显示和logo样式'
```

```
git checkout -b feature/bjh-0410-0417-审核流程优化
```
